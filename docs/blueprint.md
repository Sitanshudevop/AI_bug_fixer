# **App Name**: CodeInsightAI

## Core Features:

- Code Input: Allows users to paste code directly into a text area.
- AI-Powered Analysis: Leverages the Gemini API to analyze the pasted code, identify bugs, and explain errors.
- Bug Explanation: The AI explains the found bugs in a clear and concise manner.
- Fixed Code Generation: Generates and returns corrected code based on the analysis.
- Code Output: Displays the AI's response (bug explanation and fixed code) in a pre-formatted text area.

## Style Guidelines:

- Primary color: Forest green (#22c55e) echoing the code-like text display color. Note: the current frontend implementation also has this color applied.
- Background color: Dark navy (#082f49), a slightly lighter variant from the current design which is completely desaturated, for contrast.
- Accent color: Sky blue (#7dd3fc) to highlight key interactive elements, like the 'Analyze' button.
- Body and headline font: 'Inter', a sans-serif font, provides a modern, neutral aesthetic suitable for both headings and code-related content.
- Code font: 'Source Code Pro' to make code examples and the display of bug reports monospaced and aligned.
- Simple and clean layout with a focus on readability.
- Subtle loading animation while waiting for the AI to respond.
