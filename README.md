CodeInsightAI: Intelligent Bug Fixer
Live Demo: https://aibugfixer.netlify.app

CodeInsightAI is a streamlined web application designed to help developers debug faster. By leveraging Artificial Intelligence, the platform analyzes "buggy" code snippets, provides a detailed explanation of the logic errors, and generates a clean, optimized version of the code.

🌟 Features
Instant Error Analysis: Identify syntax and logical errors in seconds.

Plain English Explanations: Get a breakdown of why your code was failing, making it a great learning tool.

One-Click Correction: Receive a fully corrected version of your code ready to be copied back into your IDE.

Minimalist UI: A distraction-free environment focused entirely on getting your code fixed.

Developer Credit: Built-in links to GitHub and LinkedIn profiles for easy networking.

🛠 Tech Stack
Frontend: React.js / Vite

Styling: Tailwind CSS

AI Engine: Google Gemini API / OpenAI API (whichever you integrated)

Deployment: Netlify

📖 How It Works
Input: Paste your problematic code into the editor.

Analyze: Click the "Analyze Code" button to send the snippet to the AI engine.

Result: Review the identified bugs and copy the corrected code snippet.
